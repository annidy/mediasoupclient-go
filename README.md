# mediasoupclient-go

I ported [libmediasoupclient](https://github.com/versatica/libmediasoupclient) to the go implementation, and use [pion](https://github.com/pion/webrtc) for WebRTC backends.


## Features
- [x] protoo protocol
- [x] cam/mic produce
- [x] file produce
- [x] consume to file
- [x] data channel
- [ ] getStats
- [ ] screen share
- [ ] simulcast
- [ ] gui
- [ ] pause/resume stream
